## 工具库

- Guava - 工具库
- Jackson - `JSON`操作（个人不推荐使用`FastJson`）
- JUnit - 单元测试
- Mockito - 单元测试

## 工具类

- `org.springframework.util.PropertyPlaceholderHelper`：用于处理占位符的替换；
- `org.apache.commons.lang3.time.StopWatch`：用于记录执行时间的工具类
